# 基于VUE和NodeJS开发的全栈商城app
本项目前后端分离开发，前端文件夹为/src
# 前端
## Project setup
```
$ cd src
$ npm install
```

### Compiles and hot-reloads for development
```
$ npm run serve
```

### Compiles and minifies for production
```
$ npm run build
```
# 后端
后端采用Koa2框架进行搭建，数据库使用MongoDB
```
$ cd backend
$ npm install
$ npm run dev
```
